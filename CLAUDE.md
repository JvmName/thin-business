# thin-business — Pebble Watchface

## Constraints

- **Never run `pebble build` directly** — use `bun run build`, which compiles pkjs TS first
- **Never reorder `messageKeys` in `package.json`** — order determines numeric IDs; reordering breaks the running app
- **Use `minutechange`, never `secondchange`** — secondchange kills battery
- **pkjs targets ES6** — change to ES5 in `tsconfig.pkjs.json` if iOS support is ever needed
- **Never edit `src/pkjs/`** — it's entirely generated by tsc; edit `src/pkjs-src/` only
- **Nominatim requires `User-Agent: thin-business-watchface`** header on geocoding requests
